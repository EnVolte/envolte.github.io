O curso de HTML & CSS tem o objetivo de auxiliar as pessoas que estão ingressando agora no mundo da programação.

O curso é gratuito, e você pode fazer a qualquer hora do dia. Os materiais utilizados (arquivos .html e .css) nas aulas também estão disponíveis para download. Se você tiver alguma sugestão para melhorar nosso curso, envie um e-mail.

Confira a grade do curso:

  • Curso de HTML & CSS (Parte 1)
    Introdução ao HTML e tags básicas.
  • Curso de HTML & CSS (Parte 2)
    Principais tags, headings, formatação de texto, comentários e estrutura básica de página.
  • Curso de HTML & CSS (Parte 3)
    Atributos do HTML, imagens, links, listas ordernadas e listas não-ordenadas, tabelas e tipos de elementos.
  • Curso de HTML & CSS (Parte 4)
    Formulários, tipos de inputs e métodos de envio.
  • Em andamento...
