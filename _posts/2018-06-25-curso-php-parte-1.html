Desde que comecei a estudar PHP, fiquei surpreso com a forma como estou aprendendo tudo com facilidade. Eu já possuia um conhecimento básico desta linguagem de programação pela experiência que tive com o Wordpress, mas não fazia ideia de como tudo funcionava antes de ingressar na aprendizagem do mesmo. Nem mesmo sabia direito o que era uma linguagem de programação, pode-se dizer.

O PHP, Hypertext Preprocessor, é uma linguagem de programação server-side muito popular, gratuita e de código aberto. Para se ter uma noção, PHP é a 8º linguagem de programação mais popular do mundo em Junho de 2018, segundo o TIOBE. Apesar de já ter alcançado posições mais populares, e de já ter sido utilizada em mais de 80% dos websites, ela ainda é muito relevante para os programadores e para o mercado de trabalho. Ela aparece atrás de linguagens como Java (a atual mais popular), Python, C++ etc. Então, sem dúvidas, é uma boa linguagem para se aprender hoje mesmo. Afinal, conhecimento nunca é demais, não é?

Uma breve lista do que se pode fazer com PHP:

  • Gerar páginas dinâmicas.
  • Criar, abrir, ler, escrever, deletar e fechar arquivos no servidor.
  • Coletar dados de um formulário HTML.
  • Adicionar, modificar e deletar informações gravadas no banco de dados.
  • Controlar acesso de usuários pelo seu programa.

Várias plataformas, incluindo Windows, Linux, Unix e Mac OS X, são capazes de executar PHP. Ele é compatível com boa parte dos servidores, como Apache, IIS, e também suporta vários banco de dados. As vantagens são grandes, apesar de possuir algumas desvantagens também. Mas isso é assunto para outro post.

Não me considero capaz de se autointitular "professor", mas decidi criar este curso básico para introduzir quem tiver interesse em aprender esta linguagem ao mundo da programação. Ensino aqui aquilo que eu sei, e se houver, em algum momento, algum equivoco de minha parte, peço que você comente aí embaixo para que eu possa corrigir o mais rápido possível.

Se você tem interesse em fazer o curso, cadastre-se aqui para que eu possa saber o alcance do mesmo.

Esta primeira parte é apenas uma introdução, para que você se familiarize com a sintaxe do PHP. Todos os scripts em PHP iniciam com < ? p h p e terminam com "? >" (sem espaços).

Agora, para iniciarmos a nossa parte prática, é preciso que você faça o download de alguns programas. O primeiro deles será um bom editor de códigos. Eu recomendo o Sublime, mas você poderá baixar aquele que te agradar mais. Com o editor em mãos, agora precisamos transformar o nosso computador em um servidor capaz de executar o PHP, porque como eu expliquei, ele é uma linguagem server-side, que funciona ao lado do servidor e precisa do mesmo para interpretar os códigos e enviar para o seu navegador apenas o HTML, CSS e Javascript. Posso recomendar o WAMP (para Windows) ou XAMPP (Windows, Linux e OS X). Faça uma breve pesquisa e descubra qual dos dois melhor te servirá. Basicamente, esses programas instalarão o Apache + PHP + MySQL (ou outro banco de dados).

Com os dois programas em mãos, inicie o Apache e abra seu editor de códigos. Cada programa possui uma pasta padrão onde os projetos serão desenvolvidos. Indentifique-as, apague o conteúdo dela (sem apagar a pasta) e vamos criar um arquivo chamado ola.php. Não se esqueça de que quando damos nomes aos arquivos, não se usa espaçamentos ou acentuações, apenas letras, números e "-" ou "_".

Se tudo estiver intalado corretamente, seu site estará funcionando em http://localhost. Lembrando que este endereço funciona apenas no seu computador.

A primeira função que usaremos é o echo, lê-se "eco", que irá reproduzir a frase que colocaremos. Uma boa dica para melhorar a sua aprendizagem é saber a tradução das palavras, já que o PHP utiliza comados em inglês.

Agora abra http://localhost/ola.php e você verá o resultado da sua função. Gostou?

Todo mundo começa com um "Olá, mundo!", então não se desespere, pois ainda há muitas coisas para aprender. Por exemplo, e se você quiser comentar o código em PHP para lembrar de algo? Você não pode simplesmente escrever em qualquer linha. Sempre que você precisar fazer comentários de apenas uma linha, utilize duas barras //. Abra o seu editor e digite a linha a seguir dentro do seu código:

//Este é um comentário

Mas e se eu precisar comentar algo em várias linhas? É simples, utilize o /* e */.
