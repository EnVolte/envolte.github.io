Mas afinal, o que são variáveis?

As variáveis são como uma espécie de "container" que armazenam dados. Uma variável em PHP começa com um $ (cifrão) e é seguida pelo nome.

$variavel = valor;

Elas seguem algumas regras:

  • Uma variável precisa começar com uma letra ou _.
  • Variáveis não podem começar com um número, mas você pode incluí-los em qualquer outra posição.
  • Variáveis só podem conter caracteres alfanuméricos e _ (A-Z, 0-9 e _)
  • As variáveis são case-sensitive, ou seja, são sensíveis a letras maiúsculas e minúsculas. A variável $nome e a várivel $NOME são consideradas diferentes.

No exemplo acima, perceba que não precisamos declarar que tipo de conteúdo estamos usando. Uma das vantagens do PHP é que ele automaticamente converte a variável para o tipo de conteúdo, dependendo do seu valor.

Constantes

Constantes são parecidas com as variáveis, exceto pelo fato de que não podem ser mudadas depois de definidas. Também devem começar com uma letra ou _. Para criar um constante, usamos a função define().

define(nome, valor, case-insensitive)

Parâmetros:

  • nome: Especifica o nome da sua constante.
  • valor: Especifica o valor da sua constante.
  • case-insensitive: Especifica se o nome da sua constante deve ser sensível a letras maiúsculas e minúsculas. O valor padrão é false (falso).

Tipos suportados pelo PHP

As variáveis podem aceitar uma variedade de tipos de dados. São elas: String, Interger, Float, Boolean, Array, NULL (vazio), Resource.

String

Uma string é uma sequência de caracteres, como "Olá, mundo!". As strings podem ser declaradas com aspas simples ('') ou duplas ("").

Você pode juntar duas strings usando o ponto (.) como um operador de concatenação. Por exemplo: echo $string1.$string2;

Interger

O Interger representa um número inteiro (sem decimais). Não pode conter pontos de decimais, pode ser positivo ou negativo.

$int1 = 10; // Número positivo
$int2 = -10; // Número negativo

Float

o float é um número que inclui um ponto decimal.

$float = 35.000;

Boolean (booleano)

Os valores booleanos representam duas possibilidades, verdadeiro ou falso. São representados, respectivamente, por true ou false

Os valores booleanos são geralmente usados para checar condições.

Variável global

As variáveis podem ser declaradas em qualquer lugar do script. O escopo da variável é a parte do script em que a variável pode ser referenciada. As variáveis mais usadas são local e global. Uma variável declarada dentro de uma função tem o escopo local, e só pode ser acessada dentro dela.

A palavra-chave global é utilizada para dar acesso a uma variável dentro de uma função, por exemplo.

Sendo assim, podemos ter acesso à variável $nome ao chamarmos a função pegarNome().

Variáveis de variáveis

Com o PHP, podemos usar uma variável para especificar outra variável. Uma variável de variável é o valor de uma outra variável sendo utilizada como o nome dela.
