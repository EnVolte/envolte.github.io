O PHP, Hypertext Preprocessor, é uma linguagem de programação server-side muito popular, gratuita e de código aberto. Para se ter uma noção, PHP é a 8º linguagem de programação mais popular do mundo em Junho de 2018, segundo o TIOBE. Apesar de já ter alcançado posições mais populares, e de já ter sido utilizada em mais de 80% dos websites no mundo inteiro, ela ainda é muito relevante para os programadores e para o mercado de trabalho. Ela aparece atrás de linguagens como Java (a atual mais popular), Python, C++ etc. Então, sem dúvidas, é uma boa linguagem para se aprender hoje mesmo. Afinal, conhecimento nunca é demais, não é?

Se você quiser saber toda a história do PHP, desde o seu surgimento, até a versão 5 (atualmente, ele se encontra na versão 7.2.7), leia este arquivo do site oficial (está em português). O Manual do PHP é uma seção rica em conteúdo, está em português, e pode te auxiliar no aprendizado. Nele, você encontra funções exemplificadas, que vão desde as mais básicas, às avançadas.

Uma breve lista do que se pode fazer com PHP:

  • Gerar páginas dinâmicas.
  • Criar, abrir, ler, escrever, deletar e fechar arquivos no servidor.
  • Coletar dados de um formulário HTML.
  • Adicionar, modificar e deletar informações gravadas no banco de dados.
  • Controlar acesso de usuários pelo seu programa.

Esta primeira parte é apenas uma introdução, para que você se familiarize com a sintaxe do PHP. Todos os scripts em PHP iniciam com < ? p h p e terminam com "? >" (sem espaços).

Agora, para iniciarmos a nossa parte prática, é preciso que você faça o download de alguns programas. O primeiro deles será um bom editor de códigos. Eu recomendo o Sublime Text, mas você deve escolher aquele que mais te agradar. Mexer com códigos quase sempre é complicado, mas ter uma boa interface pode te inspirar no desenvolvimento de qualquer projeto. Faça uma pesquisa, considere os pontos positivos e negativos de cada um, e escolha o seu novo parceiro!

Com o editor em mãos, agora precisamos transformar o nosso computador em um servidor capaz de executar o PHP, porque, como eu expliquei, ele é uma linguagem server-side, que funciona ao lado do servidor e precisa do mesmo para interpretar os códigos. O que o seu navegador recebe é apenas o HTML, CSS e Javascript interpretados. Posso recomendar o WAMP (para Windows), XAMPP (Windows, Linux e OS X) e o EasyPHP, que como o nome já diz, é fácil de instalar e usar. Novamente, faça uma pesquisa e descubra qual programa te agrada mais. Basicamente, esses programas instalarão o Apache + PHP + MySQL (ou outro banco de dados).

Com os programas em mãos, inicie o Apache e abra seu editor de códigos. Cada programa possui uma pasta padrão onde os projetos serão desenvolvidos. Indentifique-a, apague o conteúdo dela (sem apagar a pasta) e vamos criar um arquivo chamado ola.php. Não se esqueça de que quando damos nomes aos arquivos, não se usa espaços ou acentuações, apenas letras, números e "-" ou "_".

Se tudo estiver instalado corretamente, seu site estará funcionando em http://localhost, que muitas vezes pode aparecer com um endereço ip. Lembrando que este endereço funciona apenas no seu computador!

A primeira função que usaremos é o echo ( lê-se "eco"), que irá reproduzir uma frase que escreveremos. Uma boa dica para melhorar a sua aprendizagem é saber a tradução das palavras, já que o PHP utiliza comados em inglês.

Agora abra http://localhost/ola.php e você verá o resultado da sua primeira função. No vídeo, este endereço está como index.php. A única diferença que esse endereço exerce é em sua importância. Se você criar qualquer pasta na raíz do site e criar um arquivo com este nome, ele será reconhecido como o arquivo principal.

Explicando melhor...

Se criarmos uma pasta posts na raíz do projeto e adicionarmos o arquivo index.php, poderemos acessá-lo pela URL localhost/posts. No caso de outros arquivos, devemos obrigatóriamente colocar o .php no final.

Todo mundo começa com um "Olá, mundo!", então não se desespere, pois ainda há muitas coisas para aprender. Por exemplo, e se você quiser comentar o código em PHP para lembrar de algo? Você não pode simplesmente escrever em qualquer linha. Sempre que você precisar fazer comentários de apenas uma linha, utilize duas barras //. Abra o seu editor e digite a linha a seguir dentro do seu código:

//Este é um comentário

Mas e se eu precisar comentar algo em várias linhas? É simples, utilize o /* e */.
